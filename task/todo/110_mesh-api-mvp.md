# 110_mesh-api-mvp

## Goal
Добавить API для базового управления мешами.

## Scope
- Read: список мешей текущего документа, активный меш, базовые свойства.
- Write: выбрать меш, переименовать, показать/скрыть, lock/unlock.
- Безопасное выполнение в EDT.

## Deliverables
- Endpoints `GET /mesh/*`, `POST /mesh/*`.
- Контракт ошибок (нет документа, нет выбранного меша, unsupported action).
- Минимум 10 smoke-кейсов на успешные/ошибочные вызовы.
