# 160_export-api

## Goal
Добавить API для экспортных операций текущей модели/сцены.

## Scope
- Экспорт в png/psd/видео/модельные данные (по доступности команд Cubism).
- Параметры экспорта (путь, overwrite policy, форматные опции).
- Валидация путей и безопасные ошибки.

## Deliverables
- Endpoint `POST /export`.
- Поддержка async-режима с task id (`/jobs/{id}`).
- Примеры сценариев экспорта в документации.
